ABOUT ME

Who I Am

I'm Gaurab Aryal, a developer focused on building secure web applications. I combine coding skills with cybersecurity knowledge to create applications that are both functional and protected against threats.

My Journey

I started with C and C++ to learn programming fundamentals, then moved into web development with Python, PHP, and JavaScript. Alongside this, I explored cybersecurity using tools like Metasploit and Aircrack-ng, giving me insight into both building and securing applications.

My Approach

I learn by doing and believe security should be built in from the start, not added later. I focus on writing clean code that works reliably and can withstand common security challenges.
